# When Bubbles Bite

Interactive public engagement exhibit about cavitation. Live demo (balloon burst in water tank + propeller inspection) plus iPad quiz app.

**App**: [rohitwpillai.github.io/when-bubbles-bite](https://rohitwpillai.github.io/when-bubbles-bite/)

Edinburgh Science Festival 2026 (Dynamic Earth, 10-13 April) and PEE26 (July 2026).
